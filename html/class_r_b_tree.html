Подробное описание

Красно-чёрное дерево с поддержкой дубликатов ключей.

Самобалансирующееся BST. Гарантирует высоту O(log n) при любом порядке вставки. Использует сторожевой узел nil вместо nullptr для упрощения балансировки. Ключ поиска — groom_fio.

Вставляет запись в дерево с последующей балансировкой.
