Подробное описание

Хеш-таблица с разрешением коллизий методом цепочек.

Хеш-функция: полиномиальный хеш строки с основанием BASE=31. Размер таблицы CAP=1 000 003 (простое число) для равномерного распределения. Поиск: O(1) в среднем случае.

Вставляет запись в таблицу.

Если узел с таким ключом уже есть в цепочке — запись добавляется в его records. Иначе создаётся новый узел в начале цепочки. Если слот уже занят другим ключом — счётчик коллизий увеличивается на 1.

recЗапись для вставки.
